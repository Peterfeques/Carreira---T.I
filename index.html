Sobre a Profissão

O desenvolvedor web cria e mantém sites e aplicações para a internet, como blogs, lojas virtuais e sistemas online. Atua tanto no Front-end quanto no Back-end.

  • HTML, CSS e JavaScript

  • Controle de versão com GitHub

Dicas e Tendências

  • Criar projetos para portfólio
  • Aprender frameworks modernos
  • Usar Git e GitHub
